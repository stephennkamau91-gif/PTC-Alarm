# PTC Weekly Alarm

Android application for managing weekly alarm schedules.

## Features
- Weekly alarm management
- Customizable alarm schedules
- Android alarm receiver integration

## Project Structure
- `app/` - Main application module
- `app/src/main/java/com/ptcweeklyalarm/` - Application code
- `app/src/main/res/` - Resources (drawables, layouts, values)

## Building
```bash
./gradlew build
```

## Running
```bash
./gradlew run
```
